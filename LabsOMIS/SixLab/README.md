# Интеллектуальная система автоматического создания и оптимизации веб-сайтов

Система для автоматизированной генерации веб-сайтов с учетом целей и потребностей пользователя, их последующей оптимизации для повышения производительности, удобства использования, поисковой видимости и конверсии.

## Структура системы

Система состоит из следующих подсистем:

1. **Подсистема генерации веб-сайтов** - автоматически создает прототип или готовый сайт на основе вводимых данных
2. **Подсистема хранения и обработки данных** - хранит настройки, шаблоны, контент, пользовательские данные
3. **Подсистема анализа и оптимизации** - проводит аудит скорости, юзабилити, SEO
4. **Подсистема поддержки принятия решений** - генерирует рекомендации по улучшению сайта
5. **Интерфейсная подсистема** - предоставляет консольный интерфейс для работы с системой

## Архитектура

Система построена по принципу MVC (Model-View-Controller) согласно диаграммам классов из ЛР4:

### Модели (models/):
- **User** - пользователи системы (Заказчик, Администратор)
- **Website** - веб-сайты
- **WebsiteTemplate** - шаблоны веб-сайтов
- **AnalysisResult** - результаты анализа
- **OptimizationRecommendation** - рекомендации по оптимизации
- **BusinessGoal** - бизнес-цели (третье направление развития)
- **OptimizationHistory** - история изменений оптимизации

### Контроллеры (controllers/):
- **WebsiteGenerator** - генерация веб-сайтов
- **WebsiteAnalyzer** - анализ и оптимизация
- **DecisionSupport** - поддержка принятия решений
- **MonitoringController** - мониторинг и аналитика

### Представление (views/):
- **CLIInterface** - консольный интерфейс (альтернативный способ доступа)
- **Веб-интерфейс** - Flask приложение с HTML шаблонами (основной интерфейс)

### Инфраструктура (infrastructure/):
- **DataStorage** - хранение и обработка данных (JSON)
- **OllamaClient** - клиент для работы с AI (gemma3)

## Установка

1. Установите зависимости:
```bash
pip3 install -r requirements.txt
```

2. (Опционально) Убедитесь, что Ollama запущен и модель gemma3 доступна:
```bash
ollama serve
ollama pull gemma3
```

## Использование

### Веб-интерфейс (рекомендуется)

Запустите веб-сервер:
```bash
python3 app.py
```

Откройте браузер и перейдите по адресу: `http://localhost:8000`

### CLI интерфейс (альтернатива)

Запустите консольный интерфейс:
```bash
python3 main.py
```

### Основные функции (согласно сценариям из документации):

1. **Создание веб-сайта** (Сценарий автоматической генерации)
   - Выберите тип сайта (корпоративный, интернет-магазин, блог)
   - Укажите название, описание и целевую аудиторию
   - Система автоматически сгенерирует HTML и CSS с адаптивным дизайном

2. **Анализ веб-сайта** (Сценарий оптимизации)
   - Система анализирует производительность, UX, SEO и конверсию
   - Выявляет проблемы и генерирует рекомендации
   - Формирует три сценария оптимизации: технический, пользовательский, маркетинговый

3. **Просмотр и применение рекомендаций**
   - Просматривайте рекомендации по оптимизации
   - Применяйте рекомендации для улучшения сайта
   - Система автоматически применяет выбранные изменения

4. **Мониторинг и аналитика** (Сценарий маркетинговой аналитики)
   - Общая аналитика по всем сайтам
   - Метрики конкретного сайта
   - Поведенческие метрики
   - Аналитические отчеты

5. **A/B тестирование** (Сценарий A/B-тестирования)
   - Предложение вариантов тестирования
   - Оценка эффективности улучшений

6. **Экспорт веб-сайта**
   - Экспортируйте готовый сайт в папку exports/

## Хранение данных

Данные хранятся в папке `data/` в формате JSON:
- `users.json` - пользователи
- `websites.json` - веб-сайты
- `templates.json` - шаблоны
- `analyses.json` - результаты анализов
- `recommendations.json` - рекомендации

## Интеграция с AI

Система может использовать Ollama с моделью gemma3 для:
- Генерации контента при создании сайта
- Создания дополнительных рекомендаций по оптимизации

Если Ollama недоступен, система будет работать в базовом режиме без AI-функций.

## Примеры использования

### Создание корпоративного сайта:
1. Запустите `python3 main.py`
2. Выберите "1. Создать новый веб-сайт"
3. Введите название: "Моя компания"
4. Выберите тип: "1. Корпоративный сайт"
5. Введите описание и целевую аудиторию (опционально)

### Анализ и оптимизация:
1. Создайте сайт или выберите существующий
2. Выберите "3. Анализировать веб-сайт"
3. Введите ID сайта
4. Просмотрите результаты анализа
5. Выберите "4. Просмотреть рекомендации"
6. Примените рекомендации через "5. Применить рекомендацию"
